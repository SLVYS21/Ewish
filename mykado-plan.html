0 Vue d'ensemble — flux du produit

Le produit s'organise sur 3 axes : quatre espaces utilisateur, trois briques produit (avec un module transversal cadeau), et un moteur de découverte (backgrounds, GIFs, IA).

1 · Espaces user
Créer / Recevoir / Contribuer / Business
2 · Briques produit
Carte · Mur · Cadeau (transversal)
3 · Moteurs
Backgrounds · GIFs · IA · Business Auto

Relations principales

Le mur → peut recevoir un Cadeau attaché (pot commun)
La carte → peut recevoir un Cadeau attaché (mono-cadeau)
Business → génère automatiquement des Murs (birthday tracker)
Cadeau → peut être standalone (sans carte/mur associé)
Toutes les briques → puisent dans Moteur Backgrounds
Mur & Carte → exploitent le GIF picker (curé + Giphy)
Toutes les briques → exposent des features IA (musique, voix, génération, écriture)
Mur → découverte TikTok-scroll + exports PDF/vidéo

5 GIFs & médias — bibliothèque curée d'abord

Le contributeur voit d'abord une sélection de qualité classée par catégorie. Il ne va sur Giphy qu'en fallback.

Bibliothèque myKado curée

12 catégories × 20 GIFs pré-choisis (qualité, bon goût, cohérence culturelle)
Anniv Célébration Cœur Merci Bravo Bienvenue Départ Gâteau Ballons Danse Rire Câlin

Process

  • Sélection manuelle initiale (curation admin)
  • Chaque GIF stocké sur Cloudinary (perf)
  • Tags cross-occasion (un GIF peut apparaître dans 2 cat)

Fallback Giphy

  • Bouton "Chercher plus" en bas de la biblio myKado
  • Ouvre un search Giphy intégré (API officielle)
  • Preview avant sélection
  • Au clic → rapatriement automatique Cloudinary
  • Suggestion future : proposer d'ajouter le GIF sélectionné à la biblio curée (admin review)

Autres médias

  • Audio : upload OU enregistrement micro direct
  • Vidéo : upload OU enregistrement webcam direct
  • Compression client avant upload (browser-image-compression)
  • Stickers auto-générés depuis photo IA

6 myKado for Business — automatisations équipe

Espace dédié RH / manager. Enregistrement des dates clés de l'équipe → génération auto des murs sans que le fêté ne le voit venir.

Flow d'automatisation (par milestone)

  1. Admin RH importe l'équipe (CSV / Slack / HRIS) avec dates clés
  2. Configure les règles par type de milestone (template, J-X, deadline)
  3. J-X : le système crée automatiquement le mur (background + cover + config)
  4. Envoi automatique du lien à toute l'équipe sauf au fêté
  5. Relance auto J-3, J-1 aux non-contributeurs
  6. J+0 : livraison au fêté (email / Slack notif) → mur cinématique
  7. Post-événement : dashboard analytics + option d'export

Types de milestones supportés

Anniversaire Ancienneté Welcome Farewell Promotion Retraite Naissance enfant Mariage Condoléances Récompense trimestrielle

Features complémentaires Business

  • Brand Media library (logo, backgrounds, signatures)
  • Intégration Slack / Teams / WhatsApp Business
  • Compagnon rédaction IA (suggère messages non-génériques)
  • Cadeaux corporate en masse (mono / bundle)
  • Analytics : taux participation, engagement, chaleur des messages
  • RGPD : consentement onboarding, purge à la sortie
  • Rôles : owner / manager / viewer
  • Facturation par siège actif

8 Innovations IA — pistes à explorer

Enterrer le paper-fold classique. Six catégories pour révolutionner l'expérience carte / mur / cadeau.

Nouveaux formats de carte

  • Carte-scène : zoom dans un décor animé où les éléments racontent le message
  • Carte-parcours : le destinataire "marche" dans un couloir de souvenirs
  • Carte-vinyl : disque qui tourne, aiguille tombe, chanson custom
  • Carte-story format 9:16 tap-to-progress (Instagram-like)
  • Carte-jeu : mini puzzle / quiz pour déverrouiller le message
  • Carte-AR : scan QR → apparition en réalité augmentée
  • Carte-3D scène navigable (Three.js)

Musique personnalisée

  • Suno / Udio : chanson custom depuis un prompt "jazz 60 ans maman, thème voyage"
  • Voice cloning (ElevenLabs) : voix du créateur dit le message
  • Lipsync photo (D-ID / HeyGen) : photo animée qui chante
  • Karaoké mur : chaque contributeur enregistre 1 ligne d'une chanson
  • Bibliothèque royalty-free classée par ambiance / tempo

Génération de contenu

  • Ghost writer : 3 versions du message selon relation + ton
  • Photo → animation : parallaxe, clignement, sourire (Runway / Pika)
  • Photo → vidéo 5s cinématique (Runway Gen-3, Sora)
  • Sticker generator : découpe visage → sticker animé
  • Auto-collage : N photos → composition esthétique pour cover mur

Expérience destinataire

  • Emotion detection (webcam) : reveal adapté (confetti si sourire)
  • Reveal collectif synchrone (room + proches connectés)
  • Voice-over IA : le mur "lu" par une voix au choix
  • Version podcast : export audio de tout le mur
  • Traduction auto multilingue (FR → EN si préf destinataire)

Business & automations

  • HR bot Slack : rappelle milestones + crée le mur en 1 clic
  • Compagnon rédaction : suggère messages non-génériques basés sur rôle
  • Photo-book annuel : compilation auto des murs / employé
  • Sentiment analyzer : mesure de la chaleur des messages (culture RH)

Mémoires & long-terme

  • Time capsule : mur programmé pour se déverrouiller dans X mois/années
  • Livre de vie : compilation de tous les murs reçus par une personne
  • Continuité générationnelle : mur transmis (baptême → 18 ans)
